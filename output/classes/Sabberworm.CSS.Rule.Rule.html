\Sabberworm\CSS\Rule

\Sabberworm\CSS\RuleRule

RuleSets contains Rule objects which always have a key and a value.

In CSS, Rules are expressed as follows: “key: value[0][0] value[0][1], value[1][0] value[1][1];”
